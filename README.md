# 🏓 Pong Game - JavaScript Clássico

Este é um clone do jogo Pong, um dos primeiros videogames de arcade da história. O projeto foi desenvolvido para colocar em prática conceitos de lógica de programação, manipulação de Canvas HTML5 e detecção de colisões.

## 🎮 Como Jogar
O objetivo é simples: não deixe a bola passar pelo seu paddle (barra) e tente marcar pontos contra o computador.

* **Mover para Cima:** Seta para Cima (ArrowUp)
* **Mover para Baixo:** Seta para Baixo (ArrowDown)

---

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura e elemento `<canvas>` para renderização gráfica.
* **CSS3:** Estilização básica e centralização do jogo.
* **JavaScript (ES6):** Lógica do jogo, movimentação da bola e inteligência artificial do oponente.

---

## ✨ Funcionalidades do Projeto
- **Game Loop:** Implementado com `requestAnimationFrame` para uma movimentação fluida de 60 FPS.
- **Inteligência Artificial:** O paddle do computador segue a posição vertical da bola automaticamente.
- **Sistema de Colisão:** Lógica matemática para detectar quando a bola atinge as bordas ou os paddles.
- **Reset Automático:** A bola volta ao centro sempre que alguém marca um ponto.

---

## 🧠 Aprendizados Técnicos
Durante o desenvolvimento deste projeto, pude aprender:
1. Como gerenciar coordenadas `(x, y)` em um plano 2D.
2. Como escutar eventos de teclado (`keydown` e `keyup`) para controlar elementos na tela.
3. A importância de separar a função de **Atualização de Dados** da função de **Desenho Visual**.
